## ğŸ’» Getting Started

To run this project locally:

### 1. Clone the repository:

```bash
git clone https://github.com/meetbhavsar99/EcoEats.git
cd EcoEats
```

### 2. Set up Python environment:

```bash
python -m venv env
source env/bin/activate # On Windows: env\Scripts\activate
```
